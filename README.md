- Install JDK (recommend JDK >=11)
- Install Chrome Browser, Edge Browser, Firefox Browser
- Run well on the **Windows** platform
- Setup **Allure ENV**:
  https://mvnrepository.com/artifact/io.qameta.allure/allure-java-commons
